一、安装xrdp

Ubuntu 16.04只需安装xrdp即可。

Ubuntu 14.04还需安装vnc4server

此时使用客户端连接，应该能够连接成功。

Ubuntu 16.04出现命令行画面。

Ubuntu 14.04画面为空白。
